# useful-skills

A collection of useful Claude Code skills.

## Skills

### xiaohongshu-generator
小红书内容生成器 - 通过访谈式对话引导用户完善小红书内容创意，最终生成符合小红书风格的发布内容。

### sz-housing-matcher
深圳市保障房政策追踪与智能匹配助手 - 自动搜索最新政策并按个人情况排序，包含交通便利性分析。支持安居房、人才房、公租房等多种房源类型的智能匹配。

## Usage

这些skills是为Claude Code设计的。将skill文件夹复制到你的 `~/.claude/skills/` 目录即可使用。
